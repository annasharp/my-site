# Мой сайт

Простой сайт на HTML/CSS/JS с сеткой карточек, которые можно добавлять и удалять.

## Структура проекта

```
my-site/
├── index.html      — разметка страницы
├── css/
│   └── style.css   — стили, включая сетку (CSS Grid)
├── js/
│   ├── data.js     — начальные данные карточек (редактируй их тут)
│   └── script.js   — логика рендера, добавления и удаления карточек
└── README.md
```

## Как управлять карточками

Два способа:

1. **Через сайт** — форма наверху страницы добавляет карточку в текущую сессию.
   Кнопка «Удалить» на карточке убирает её. Эти изменения пропадают при
   перезагрузке страницы — это демо-режим, без сервера и базы данных.

2. **Навсегда** — открой `js/data.js` и добавь/удали объект в массиве
   `initialItems`. Сохрани файл — при следующей загрузке страницы карточка
   останется навсегда (пока ты сам её не уберёшь).

Если позже понадобится, чтобы карточки, добавленные через форму, тоже
сохранялись навсегда (например, между посещениями с разных устройств),
это уже потребует базы данных и бэкенда — дай знать, помогу с этим отдельно.

## Запуск в VS Code

1. Установи [VS Code](https://code.visualstudio.com/), если ещё не установлен.
2. Открой папку `my-site` через **File → Open Folder…**.
3. Установи расширение **Live Server** (автор: Ritwick Dey) через вкладку
   Extensions (значок квадратиков слева).
4. Кликни правой кнопкой по `index.html` → **Open with Live Server**.
   Сайт откроется в браузере и будет обновляться при каждом сохранении файла.

## Как загрузить проект на GitHub

### Шаг 1 — установи Git (если ещё нет)
Проверь в терминале VS Code (`` Ctrl+` ``):
```bash
git --version
```
Если команда не найдена — скачай Git с https://git-scm.com/downloads

### Шаг 2 — создай репозиторий на GitHub
1. Зайди на https://github.com → **New repository**.
2. Дай имя, например `my-site`. Не добавляй README/.gitignore на этом
   шаге — они уже есть у нас локально.
3. Нажми **Create repository** — GitHub покажет команды для загрузки,
   похожие на те, что ниже.

### Шаг 3 — инициализируй Git локально и запушь
В терминале VS Code, находясь внутри папки `my-site`:

```bash
git init
git add .
git commit -m "Первый коммит: базовая структура сайта"
git branch -M main
git remote add origin https://github.com/ТВОЙ_НИК/my-site.git
git push -u origin main
```

После этого файлы появятся в репозитории на GitHub.

### Дальнейшие изменения
Каждый раз, когда что-то меняешь и хочешь сохранить это на GitHub:

```bash
git add .
git commit -m "Описание того, что изменил"
git push
```

## Публикация сайта (бесплатно, через GitHub Pages)

1. В репозитории на GitHub зайди в **Settings → Pages**.
2. В разделе **Source** выбери ветку `main` и папку `/ (root)`.
3. Сохрани — через пару минут сайт будет доступен по адресу вида
   `https://ТВОЙ_НИК.github.io/my-site/`.
